# Scales for the \*colour aesthetics

These are the various scales that can be applied to the xsidebar or
ysidebar colour aesthetics, such as xcolour and ycolour. They have the
same usage as existing standard ggplot2 scales.

## Value

returns a ggproto object to be added to a ggplot
