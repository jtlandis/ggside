# Class Definitions

This documentation provides an overview of the S7 and ggproto classes
used in the `ggside` package.

## ggproto classes

- `class_ggside_opt` is a subclass of `class_ggproto` and is more
  described in the
  [ggside-options](https://jtlandis.github.io/ggside/reference/ggside-options.md)
  documentation.

&nbsp;

- `class_ggside_layer` is a subclass of `class_ggproto` and is more
  described in the
  [ggside-layers](https://jtlandis.github.io/ggside/reference/ggside_layer.md)
  documentation.

&nbsp;

- `class_ggside_scale` is a subclass of `class_ggproto` and is more
  described in the
  [ggside-scales](https://jtlandis.github.io/ggside/reference/ggside-scales.md)
  documentation.

## S7 classes

- `class_ggside` is a subclass of [ggplot2's
  class_ggplot](https://ggplot2.tidyverse.org/reference/class_ggplot.html)
  and is used to represent a ggplot object with ggside options.
