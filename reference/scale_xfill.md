# Scales for the \*fill aesthetics

These are the various scales that can be applied to the xsidebar or
ysidebar fill aesthetics, such as xfill and yfill. They have the same
usage as existing standard ggplot2 scales.

## Value

returns a ggproto object to be added to a ggplot
